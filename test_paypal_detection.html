PayPal Phishing Detection Test

This page is designed to test the PayPal phishing detection capability.

Test Domains:

  • paypa1-login.com - Should be detected (1 instead of l)
  • paypal-security.com - Should be detected (not official domain)
  • secure-paypal.net - Should be detected (suspicious)
  • paypal.com - Should be safe (official)

What the extension should detect:

  • ✅ Character substitution (paypa1 instead of paypal)
  • ✅ HTTP protocol for financial sites
  • ✅ Login/security related keywords
  • ✅ Non-official PayPal domains

Expected behavior for paypa1-login.com:

  1. Extension detects "paypa1" variant (80 points)
  2. Extension detects HTTP protocol with "login" keyword (40 points)
  3. Extension detects mixed letters/numbers (10 points)
  4. Total risk score: 130+ (capped at 100)
  5. Should trigger blocking (threshold > 60)

Troubleshooting:

If the extension doesn't detect this site:

  1. Open Chrome DevTools (F12)
  2. Check Console tab for "Phishing Detector:" messages
  3. Verify extension is loaded and active
  4. Check if site is in whitelist
  5. Reload the extension if needed
